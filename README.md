## Prequisition

- Node 12

```bash
sudo n 12
```

## Quick Start

### 클론

```bash
git clone https://github.com/the-masked-developer/the-masked-developer.github.io.git
```

### 브랜치 변경(writing)

```bash
git checkout -b writing
```

### 설치

```bash
npm install
```

### 문서 업로드

`/source/_posts` 경로에 마크다운(`.md`) 파일 업로드

### Deploy 명령어 실행

```bash
npm run deploy
```

### 로컬 테스트

```bash
npm run serve
```
